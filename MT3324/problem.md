# MT3324：AND 子数组求和

## 题目描述
给出一个长度为 `n` 的数列 `a`。定义

$$
 p_{i,j} = a_i \; \text{AND} \; a_{i+1} \; \text{AND} \; \cdots \; \text{AND} \; a_j
$$

其中 `AND` 表示按位与运算。请计算：

$$
\sum_{i=1}^{n} \sum_{j=i}^{n} p_{i,j}
$$

也就是所有连续子数组按位与结果的总和。

## 输入格式
- 第一行一个整数 `n`，表示数列长度；
- 第二行 `n` 个整数，表示数列 `a`。

## 输出格式
- 一个整数，表示答案。

## 样例
**输入：**
```
5
5 8 4 5 6
```

**输出：**
```
40
```

## 数据范围
- $1 \le n \le 100000$
- $0 \le a_i \le 10^9$

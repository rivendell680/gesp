# MT3324：最大化购买甜品后的中位数甜度

## 题目描述
你有 `v` 的预算，需要从 `n` 个甜品中购买恰好 `m` 个。第 `i` 个甜品有：
- 甜度 `a_i`
- 价格 `b_i`

目标是在总价格不超过 `v` 的前提下，让买到的 `m` 个甜品的**中位数甜度尽可能大**。

> 本实现按常见竞赛判定方式处理：中位数位置为升序后第 `(m+1)//2` 个元素（即至少有 `(m+1)//2` 个元素不小于该值）。

## 输入格式
- 第一行三个整数 `v n m`
- 接下来 `n` 行，每行两个整数 `a_i b_i`

## 输出格式
- 输出一个整数，表示可达到的最大中位数甜度。

## 数据范围
- `1 <= m <= n <= 1e5`
- `1 <= a_i <= 1e9`
- `1 <= b_i <= v <= 1e9`

## 样例
输入：
```text
20 5 3
3 5
5 6
8 7
10 6
15 10
```

输出：
```text
8
```
